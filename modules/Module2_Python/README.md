# Module 2: Data, Datasets & AI Ethics

This module focuses on how data powers AI systems and introduces ethical concepts such as bias, security, privacy, and fairness.

## Topics Covered

- What datasets are and how they are structured
- Data cleaning and preprocessing basics
- Common dataset formats (CSV, JSON, images)
- Bias in machine learning datasets
- Ethical and responsible AI considerations

## Assignments in This Module

- Dataset analysis worksheet
- Ethical case study reflection
- Capstone: Identify the data your project requires

## Capstone Project Connection
For the capstone, you:
- Determine your dataset
- Explain ethical concerns related to your topic
